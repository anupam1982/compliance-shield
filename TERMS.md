# Terms of Service

Compliance Shield is provided as an open-source GitHub App for detecting security and compliance violations in repositories.

## Usage

Users may install and use the application to analyze their repositories for potential security issues.

## Limitations

Compliance Shield provides automated analysis and recommendations. It does not guarantee the absence of vulnerabilities or compliance violations.
