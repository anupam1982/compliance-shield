# Privacy Policy

Compliance Shield is a GitHub App that analyzes repository content to detect security and compliance violations.

## Data Usage

The application reads repository files only for the purpose of identifying:

- exposed secrets
- insecure code patterns
- compliance violations

No repository content is stored outside GitHub infrastructure.

## Data Storage

Compliance Shield may store minimal scan metadata inside the repository to maintain scan history. This metadata does not include source code or sensitive repository content.

## Third-Party Data Sharing

Compliance Shield does not transmit repository data to external services.

## Contact

For questions about this privacy policy, contact the project maintainer via the GitHub repository.
